# armina
search from terminal 

**Google**   
```bash
armina -g <search promt> 
```
**Youtube**   
```bash
armina -y <search promt> 
```
**https**  
```bash
armina -w <website> 
```
**Twitch**
```bash
armina -t <channel>
```

**DuckDuckGo**
```bash
armina -d <search promt>
```

## example
```armina -g what is rust?```
