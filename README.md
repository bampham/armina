# armina
Search from terminal which allows you to open new tab in defualt browser at specified addr/promt,

**Google**   
```bash
armina -g <search promt> 
```
**Youtube**   
```bash
armina -y <search promt> 
```
**https**  
```bash
armina -w <website> 
```
**Twitch**
```bash
armina -t <channel>
```

**DuckDuckGo**
```bash
armina -d <search promt>
```
